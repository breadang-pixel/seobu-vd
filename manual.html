📊 ③ 탭별 주요 기능
📊
탭 1. 팀별 현황
  • 팀별 등급 분포 차트 — 각 팀의 S~D 등급 구성 비율을 막대그래프로 시각화
  • 등급 하락 TOP 5 매장 — 전월 대비 등급이 실제로 하락한 매장 중 점수 낙폭 상위 5개핵심
  • 경계선 매장 분석 — 다음 등급까지 근접한 매장 목록 (승급 유망 / 강등 위험)
  • 팀 취약 항목 — 팀별로 평균 점수가 낮은 평가 항목 자동 파악
  • 등급 현황 수치 (S/A/B/C/D 각 개수) 표 제공
⚠️ 등급 하락 기준: 점수만 낮아진 경우는 포함되지 않습니다. 등급이 실제로 하락(예: A→B)한 매장만 표시됩니다.
🏢
탭 2. 대리점별 현황
  • 대리점 목록 테이블 — 팀명, 대리점명, 매장수, 등급, 수수료 요약
  • 대리점명 클릭 → 💰 VD수수료 계산기 팝업 실행NEW
  • 매장보기 버튼 — 해당 대리점의 매장 목록 상세 표시
  • 분석결과 버튼 — AI 분석 결과 및 개선 제안 표시
  • 수수료 합계 기준 정렬 및 등급별 색상 구분
💰 VD수수료 계산기 사용법

대리점명을 클릭하면 해당 대리점 전체 매장의 수수료 시뮬레이션이 열립니다.
각 매장별로 미달성 항목을 클릭하면 달성 여부를 토글하고, 등급·수수료 변화를 실시간 예측할 수 있습니다.
Before / After 비교로 수수료 증감 효과를 즉시 확인하세요.

🏪
탭 3. 매장별 현황
  • 등급별 서브탭 — S / A / B / C / D 탭으로 구분하여 탐색
  • 매장별 총점, 항목별 세부점수, 등급, 소속 대리점 표시
  • 경계선 매장 — 승급 근접 매장과 강등 위험 매장 별도 구분
  • 항목별 달성·미달성 여부 색상 표시
  • 필터(팀/대리점/등급)와 연동되어 조건에 맞는 매장만 표시
📋
탭 4. 상세 리스트
  • 전체 매장 데이터 테이블 — 모든 평가 항목 점수 포함
  • Excel 다운로드 — 현재 필터 조건 그대로 엑셀 파일 저장
  • PDF 다운로드 — 인쇄용 PDF 파일 생성
  • 컬럼 정렬 기능으로 원하는 기준 정렬
  • 필터와 연동 — 필터 변경 시 테이블 즉시 반영
📢
탭 5. 공지사항
  • 관리자가 등록한 공지·안내사항 목록 표시
  • 공지 등록 ADMIN — 제목 + 내용 입력 후 저장
  • 공지 삭제 ADMIN — 개별 공지 삭제
  • Firebase 연동 시 등록 즉시 모든 사용자에게 실시간 노출SYNC
  • 최신순 정렬, 등록 일시 표시
📈
탭 6. 활용현황 ADMIN
  • 관리자 전용 탭 — 일반 사용자 접속 시 탭 비표시
  • 일별·월별 조회수 추이 차트
  • 접속 기기 유형 분포 (PC / 모바일)
  • 최근 접속 기록 목록 (날짜, 시간)
  • Firebase에 조회 로그가 저장되어 누적 통계 유지SYNC

  • 관리자 비밀번호 변경 — 관리자 접속 비밀번호 변경

  • 현재 데이터 확인 — Firebase에 저장된 최신 데이터 조회

🔥 ⑤ Firebase 실시간 연동
🔥 Firebase Firestore 연동 안내

  • 공지사항 실시간 공유 — 등록 즉시 전체 사용자 화면에 노출
  • 비밀번호 클라우드 저장 — URL이 달라도 동일 비밀번호 유지
  • 조회 로그 기록 — 접속자 통계 자동 수집
  • Firebase 설정 완료 후 자동 연결 (재입력 불필요)
  • 오프라인 상태에서도 마지막 캐시 데이터로 조회 가능
  • Firebase 프로젝트: seobu-vd
  • Firestore 규칙: 읽기 허용 / 쓰기 인증 필요
